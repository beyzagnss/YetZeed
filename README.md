# **YetZeed 🌱**

Evindeki boş alanı, AI destekli akıllı bir üretim merkezine dönüştür.

## **Problem**

Modern şehir hayatında taze ve sürdürülebilir gıdaya erişim zorlaşırken, bireyler evlerindeki atıl alanları ekonomik bir değere dönüştürmek için gereken teknik bilgi ve zaman yönetimi becerisine sahip değil. Geleneksel tarım yöntemleri, yoğun iş temposuna ve ev yaşamının dinamiklerine (ebeveynlik, esnek çalışma vb.) uyum sağlamakta yetersiz kalıyor.

## **Çözüm**

YetZeed, evdeki boş alanları AI desteğiyle yüksek verimli birer üretim merkezine dönüştüren hibrit bir asistandır. Gemini 1.5 Flash API ile mekan analizi yaparak herkese en karlı ürünleri önerir ve kişiselleştirilmiş bir üretim takvimi oluşturur. Özel bir misyon olarak; kadın girişimcilerin "görünmez emek" yükünü hafifletmek için üretim rutinlerini bebek uyku saatleri gibi hassas takvimlere göre optimize eder ve kadınlara özel devlet teşviklerini (KOSGEB/TKDK) öncelikli olarak filtreler.

## **Kullanılan Teknolojiler**

**Yapay Zeka ve Geliştirme:**

* 🧠 **Gemini 1.5 Flash API** (Google AI Studio) — Mekan analizi, dinamik takvimleme ve hibe eşleştirme  
* 🤖 **Claude, ChatGPT & Perplexity** — Vibe coding felsefesiyle kod yazımı, hata düzeltme ve pazar araştırması  
* 🖌️ **React \+ Tailwind CSS** — Modern, hızlı ve responsive kullanıcı arayüzü  
* 🚀 **Cursor (Agent Mode)** — AI destekli kod editörü  
* 🌐 **Netlify** — Canlıya alma ve küresel erişim

**Fiziksel İnovasyon:**

* 🌌 **Antigravity** — Dikey tarım ünitelerinin verimliliğini artırmak ve alan tasarrufunu maksimize etmek için geliştirilen fiziksel inovasyon konsepti *(Demo sürümünde simüle edilmiştir)*

## **Nasıl Çalıştırılır?**

1. **Depoyu klonlayın:**

bash  
  git clone https://github.com/beyzagnss/YetZeed.git  
   cd YetZeed

2. **Bağımlılıkları yükleyin:**

bash  
  npm install

3. **API anahtarını ekleyin:**

bash  
  \# .env dosyası oluşturun  
   VITE\_GEMINI\_API\_KEY\=ANAHTARINIZ

4. **Uygulamayı başlatın:**

bash  
  npm run dev

Tarayıcınızda `http://localhost:5173` adresini açın. 🚀
